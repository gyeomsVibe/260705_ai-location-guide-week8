# 📍 나만의 위치 안내 서비스

우리 가게, 우리 모임 장소를 쉽게 안내해 주는 아주 간단한 웹 서비스입니다.
Node.js와 Express로 만든 8주차 실습 프로젝트입니다.

> 🗺️ 지도(카카오맵)는 배포 후 연결할 예정입니다. 지금은 안내 문구만 표시됩니다.

---

## 🖥️ 화면 미리보기

접속하면 아래 내용이 보입니다.

- 제목: **나만의 위치 안내 서비스**
- 설명 문구
- **지도는 배포 후 연결합니다** 안내 상자

---

## 🚀 실행 방법 (초보자용)

### 1. 준비물
- [Node.js](https://nodejs.org) 설치 (설치 확인: 터미널에 `node -v` 입력 시 버전이 나오면 OK)

### 2. 패키지 설치
프로젝트 폴더에서 터미널을 열고 아래 명령을 입력합니다. (처음 한 번만 하면 됩니다.)

```bash
npm install
```

### 3. 서버 실행

```bash
npm start
```

터미널에 `서버가 실행되었습니다!` 메시지가 보이면 성공입니다.

### 4. 브라우저에서 접속
아래 주소로 접속합니다.

```
http://localhost:3000
```

### 5. 서버 종료
터미널에서 `Ctrl + C` 를 누르면 서버가 꺼집니다.

## 📁 파일 구조

```
.
├── server.js          # 웹 서버 (포트 3000)
├── public/
│   └── index.html     # 브라우저에 보이는 화면
├── package.json       # 프로젝트 정보 & 의존성 목록
├── package-lock.json  # 설치된 패키지 버전 고정 파일
└── .gitignore         # Git에 올리지 않을 파일 목록
```
