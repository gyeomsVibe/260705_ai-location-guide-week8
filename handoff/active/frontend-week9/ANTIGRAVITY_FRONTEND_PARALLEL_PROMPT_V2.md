## 용어 정정

이전 계획의 관련 표현은 모두 `당근(밴드)`로 정정한다.

`당근(밴드)`는 특정 서비스의 데이터나 화면을 복제한다는 뜻이 아니다. `당근(밴드)`의 지역 기반성과 친목모임 성격을 결합한 참고 개념이며, 실제 사용자 화면에는 타사 브랜드를 노출하지 않고 `동네 모임` 또는 `지역 친목모임`으로 표현한다.

공공서비스 혜택 기능을 유지하면서, 사용 빈도가 높은 지역 생활·커뮤니티 추천을 제품의 기본 경험으로 강화합니다.

핵심 기능 우선순위:

1) 벙 맛집

  - 분위기: 대화하기 좋은 / 활기찬 / 든든한 식사

- 실제 최적 동선, 이동시간, 운영시간, 예약 가능 여부가 검증되지 않았으면 확정 표현을 쓰지 마세요.
