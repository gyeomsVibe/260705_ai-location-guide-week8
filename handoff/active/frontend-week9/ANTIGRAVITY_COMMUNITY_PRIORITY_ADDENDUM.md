# Antigravity 추가 지시문 — 지역 커뮤니티·일상 추천 우선순위

이 문서는 `ANTIGRAVITY_BENEFITS_PARALLEL_PROMPT.md`를 대체하지 않고 제품 우선순위를 보강한다. 충돌할 때는 이 문서의 사용자 경험 우선순위를 적용한다.

## 제품 방향

공공혜택은 중요한 별도 기능이지만 기존 생활 검색을 대체하지 않는다. 앱의 기본 진입과 가장 자주 쓰는 기능은 `동네에서 지금 무엇을 할지 정하기`다.

우선순위는 다음과 같다.

1. 벙 맛집
2. 점심 뭐 먹지
3. 저녁 뭐 먹지
4. 지역 데이트 코스
5. 지역 헬스장
6. 산책로·카페·맛집·일반 장소 검색
7. 정부 혜택 탐색

## 용어 정의

### 벙 맛집

`벙 맛집`은 음식 종류만 찾는 검색이 아니다. 지역 친목모임이 실제 모임 장소를 결정하도록 돕는 맛집 추천 흐름이다.

필요 조건:

- 모임 지역
- 인원수
- 1인당 예산
- 점심·저녁·야식 시간대
- 조용한 대화·활기찬 분위기·회식 등 모임 성격
- 주차 필요 여부
- 대중교통 접근성
- 음식 선호와 제외 음식

초기 버전에서 제공 데이터로 확인할 수 없는 `예약 가능`, `단체석 보장`, `평점 우수`, `현재 영업 중`은 단정하지 않는다. 후보를 보여주고 카카오맵 상세 확인과 전화 확인 행동으로 연결한다.

### 뭐 먹지 시리즈

사용자가 검색어를 생각하지 않아도 다음 질문만으로 후보를 좁힌다.

- 언제: 점심 / 저녁 / 야식
- 누구와: 혼자 / 둘 / 3~4명 / 5명 이상
- 기분: 든든하게 / 가볍게 / 매콤하게 / 따뜻하게 / 새로운 메뉴
- 예산: 1만원 이하 / 1~2만원 / 2만원 이상

초기 버전은 현재 위치 주변 카카오 장소 후보를 사용한다. 같은 결과 반복을 줄이기 위해 최근 선택은 브라우저 로컬 저장소에만 보관할 수 있다.

### 지역 데이트 코스

한 장소 검색이 아니라 2~3개 장소를 순서대로 고르는 코스 구성 기능이다.

- 식사
- 카페 또는 디저트
- 산책·공원·전시·영화 중 한 가지

초기 버전은 `후보 코스`로 표시하며 실제 이동시간, 운영시간, 예약 가능 여부를 보장하지 않는다. 각 장소의 카카오맵 상세 페이지와 길찾기로 연결한다.

### 지역 헬스장

초기 데이터로 검증 가능한 항목은 위치, 거리, 상호, 주소다. 가격, 시설 품질, 혼잡도, 이용자 평가는 현재 공공 API와 카카오 키워드 검색만으로 확인할 수 없다.

따라서 초기 화면은 `가까운 헬스장 후보`로 표시하고, 향후 지역 사용자 평가가 쌓인 뒤에만 `평가 좋은 헬스장` 순위를 제공한다.

## 프론트엔드 필수 보존 사항

- 기존 자유 검색창과 카카오 키워드 검색을 삭제하지 않는다.
- 기존 카페 1km·3km·5km 반경 검색을 유지한다.
- 카페, 맛집, 병원, 약국, 은행, 마트, 공원, 등산 카테고리를 누락하지 않는다.
- 공공혜택 탭을 추가하더라도 `내 주변 찾기`를 기본 선택 상태로 유지한다.
- 당근(밴드) 서비스의 데이터, 화면, 브랜드 또는 비공개 API를 복제·스크래핑하지 않는다.

## 이번 병렬 작업 범위

현재 Antigravity가 혜택 UI를 수정 중이면 같은 파일의 대규모 구조 변경을 추가하지 않는다. 우선 다음만 반영하거나 Codex에게 충돌 위험을 보고한다.

1. `내 주변 찾기`가 기본 탭인지 확인한다.
2. 기존 카테고리와 자유 검색이 모두 남아 있는지 확인한다.
3. 향후 추가할 주요 진입점의 정보구조를 제안한다.
   - `벙 맛집`
   - `점심 뭐 먹지`
   - `저녁 뭐 먹지`
   - `데이트 코스`
   - `헬스장`
4. 아직 구현하지 않은 추천 품질이나 사용자 평점을 가짜 데이터로 표시하지 않는다.

## 후속 구현 단계

### 1단계 — 검색 프리셋

기존 카카오 장소검색을 재사용해 벙 맛집, 점심, 저녁, 헬스장, 산책로 후보를 제공한다.

### 2단계 — 코스 조합

선택한 지역에서 식사·카페·산책 후보를 조합하고 사용자가 직접 교체할 수 있게 한다.

### 3단계 — 커뮤니티 평가

로그인, 신고, 차단, 개인정보 최소수집, 조작 방지 정책과 데이터 저장소가 준비된 뒤 지역별 평가·후기·모임 적합도 기능을 도입한다.

## 완료 보고에 추가할 항목

- 기존 생활 카테고리 보존 여부
- `내 주변 찾기` 기본 진입 유지 여부
- 커뮤니티 추천 기능과 공공혜택 기능이 서로 가려지지 않는지
- 현재 데이터로 검증할 수 없어 의도적으로 표시하지 않은 정보
